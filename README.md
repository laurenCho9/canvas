## Canvas API란?

- canvas html element 를 사용한다.
- canvas API는 우리가 Javascript로 그래픽을 그릴 수 있게 해준다.
  - WebGL API로 2D 그래픽이나 3D 그래픽을 그릴 수 있다. (3D 작업에 특히 빛을 발휘.)
- 노마드 코더 강의로 2D 그래픽만 연습할건데 Canvas API는 주로 2D 그래픽에 중점을 둡니다.
  - 하드웨어 가속이 된 것이다.
